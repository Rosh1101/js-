# Prompt Quizzer

This is a simple JavaScript quiz game that runs in the browser console.

The quiz asks questions using `prompt()` and shows feedback using `alert()`.
It keeps track of the score and shows the final result at the end.

## How to Run

1. Open any browser (Chrome / Edge)
2. Press F12 to open Developer Tools
3. Go to the Console tab
4. Paste the code from `quiz.js`
5. Press Enter and answer the questions

## Features

- Uses array to store questions and answers
- Uses for loop to ask questions one by one
- Converts input using `toLowerCase()` and `trim()`
- Shows correct or wrong message after each answer
- Displays final score
